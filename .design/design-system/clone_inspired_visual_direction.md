# LancerAI Clone-Inspired Visual Direction

## Identity
LancerAI should feel like a premium AI career cockpit: Linear structure, Superlist color energy, Raycast quick actions, Vercel metrics, Stripe/Framer landing rhythm, and Notion-like content clarity.

## UI UX Pro Max Output
- Product direction: AI SaaS career/interview assistant.
- Style: Aurora UI.
- Density: high dashboard density.
- Motion: standard micro-interactions.
- Primary palette direction: AI purple, indigo, generation pink, vivid cyan, accessible text contrast.

## Theme
- Dark: deep ink background, luminous aurora gradients, glassy but readable panels.
- Light: soft lavender/cool white app canvas, strong dark text, subtle aurora glows, crisp borders.
- Use semantic CSS variables for every surface: app, sidebar, topbar, cards, inputs, dropdowns, modal, table, AI panels, badges, charts.

## Feature Accents
- Voice/interview: pink-violet.
- CV/document intelligence: sky-cyan.
- Job matching: green-teal.
- Question bank: indigo.
- Reports/analytics: amber.
- Settings/profile: neutral violet.

## Components
- Cards: 8px radius, soft border, aurora highlight, hover lift via transform/box-shadow.
- Buttons: one primary CTA per screen, secondary outline, clear disabled states, 44px touch target where possible.
- Inputs: visible border, focus ring, strong label context, readable placeholder.
- Badges/chips: semantic token colors, no color-only status.
- Panels: clean header/body split, dense but breathable spacing.
- Empty states: product mockup or original CSS/SVG visual plus action.
- AI panels: assistant mark, live state pulse, explicit next action.

## Motion
- 150-300ms transitions for hover, sidebar, dropdown, modal, tab, state changes.
- Infinite animation only for recording/loading.
- Respect `prefers-reduced-motion`.

## Accessibility
- Maintain visible focus.
- Avoid emoji icons.
- Use semantic buttons/links.
- Keep contrast readable in both themes.
- Avoid horizontal overflow at 375px.
