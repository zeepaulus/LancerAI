# Framer Reference Analysis

Sources: https://www.framer.com, https://www.framer.com/community/gallery, https://www.framer.com/dictionary/framer-motion

## Layout
- Template-like rhythm with bold sections, big visual reveals, and interactive-feeling blocks.

## Navigation
- Marketing nav leads users into templates, AI builder, and examples.

## Typography
- Large expressive headlines paired with compact explanatory copy.

## Grid
- Section-based layout with generous spacing and visual rhythm.

## Components
- Template cards, gallery grids, animated previews, feature panels, builder examples.

## Cards
- Cards are visual and preview-oriented; strong imagery or mockups carry the concept.

## Hero
- Hero should show the product transformation, not just describe it.

## Motion
- Polished appear, hover, tap, transition, and component variant motion. Respect reduced motion.

## CTA
- Encourage starting/trying quickly.

## Color
- Modern high-contrast surfaces with bold accents and gradient moments.

## Icons
- Clean UI icons, often paired with previews.

## Empty State
- Should feel like a starting canvas.

## Loading
- Smooth skeleton/preview loading supports visual continuity.

## Visual Philosophy
- Creative, fluid, launch-ready, visually rhythmic.
